# Internetapplicaties - Webshop

## Express opzetten
Terminal in c:/
`npm install express-generator -g`

Terminal in project folder
`express --git --pug`
`npm install`

`SET DEBUG=*` 
`npm start`

## Automatisch herstarten na wijziging
`npm install --save-dev nodemon`
start project nu door: `npm run devstart`
